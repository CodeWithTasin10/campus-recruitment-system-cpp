# Campus Recruitment System

A C++ console-based Campus Recruitment System developed as a **2nd Semester CSE Lab Project**. The project demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts and binary file handling to automate the campus recruitment process.

## Features

- Student Registration & Login
- Company Registration & Login
- Admin Panel
- Job Posting & Management
- Job Application System
- Applicant Hiring
- Placement Report Generation
- Persistent Data Storage using Binary Files

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- Binary File Handling

## OOP Concepts

- Encapsulation
- Abstraction
- Inheritance
- Polymorphism
- Templates
- Operator Overloading
- Friend Function
- Exception Handling
