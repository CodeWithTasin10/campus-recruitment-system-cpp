# Campus Recruitment System

A C++ console-based Campus Recruitment System developed using Object-Oriented Programming (OOP).

## Features

- Student Registration & Login
- Company Registration & Login
- Admin Panel
- Job Posting
- Job Application
- Hiring System
- Report Generation
- Binary File Handling

## Technologies

- C++
- Object-Oriented Programming
- STL
- Binary File Handling

## OOP Concepts

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Templates
- Friend Function
- Operator Overloading
- Exception Handling
